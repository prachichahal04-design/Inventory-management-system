# Inventory Management System

## Description
The Inventory Management System is a web-based application developed to manage product records, stock levels, and inventory operations. The system allows users to add, view, update, and delete product information through a simple dashboard.

## Features
- Secure login system
- Admin dashboard
- Add new products
- View product list
- Edit product details
- Delete products
- Manage stock quantity
- Simple and user-friendly interface

## Technologies Used
- HTML
- CSS
- JavaScript
- PHP
- MySQL
- Apache Server
- Ubuntu/Linux

## Project Purpose
This project was created as part of my academic learning to improve my skills in web development, database management, CRUD operations, and PHP backend development.

## How to Run
1. Install Apache, PHP, and MySQL.
2. Copy the project folder into:
   /var/www/html/
3. Start Apache and MySQL:
   sudo service apache2 start
   sudo service mysql start
4. Import the database file into MySQL if provided.
5. Open the project in browser:
   http://localhost/inventory-system
